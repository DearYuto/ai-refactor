# 마켓 페이지 기능 기획 (2026-02-18)

## 기획 배경
BTC 전용 프로토타입 → 실제 거래소 서비스 점진적 확장
한국 시장 특화 (업비트/빗썸 스타일 고려)

## P0 기능 목록
1. 지정가 주문 (Limit Order)
2. 미체결 주문 관리 패널
3. 주문 취소 기능
4. 잔고 기반 최대 주문 수량 계산 (MAX 버튼)
5. 멀티 페어 지원 구조 (ETH 등 추가)

## P1 기능 목록
1. Stop-Loss / Stop-Limit 주문
2. 주문 히스토리 탭 (체결 내역 영구 보존)
3. 오더북 클릭으로 주문 가격 자동 입력
4. 차트 인터벌 확장 (4h/1d/1w)
5. 포지션 손익 실시간 표시

## 데이터 모델 변경 필요사항
- Order 상태: filled → pending/open/filled/cancelled/rejected
- OrderBook 내부 매칭 엔진 설계 필요 (P1 이후)
- DB 연동 필수 (Prisma 이미 설치됨)
